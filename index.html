ⓘ Como funciona este indicador
Para cada ano, o sistema compara o número de concluintes com o número de ingressantes de 6 anos antes (duração típica de engenharia).

Exemplo:
Formados em 2022: 50.000
Ingressantes em 2016: 200.000
Razão: 50.000 ÷ 200.000 = 0,25
O sistema repete para todos os pares possíveis (2021↔2015, 2020↔2014...), calcula a média das razões e multiplica por 100.
ⓘ Anos sem dados de ingressantes 6 anos antes são descartados. Com dados a partir de 2010, somente concluintes de 2016 em diante participam do cálculo.

ⓘ Como funciona este indicador
Mesma lógica do indicador “Concluem”: para cada ano, o sistema divide o número de novos registros no CREA pelo número de ingressantes de 6 anos antes.

Exemplo:
Novos registros em 2022: 30.000
Ingressantes em 2016: 200.000
Razão: 30.000 ÷ 200.000 = 0,15
O sistema repete para todos os pares possíveis (2021↔2015, 2020↔2014...), calcula a média e multiplica por 100.
ⓘ A defasagem de 6 anos é a mesma usada em “Concluem”, mantendo a coerência: os três indicadores se referem ao mesmo grupo hipotético de ingressantes.
% dos concluintes
Divisão direta entre os dois indicadores acima:
“Registram-se” ÷ “Concluem”.
Como ambos usam a mesma base (ingressantes de 6 anos antes), esta razão representa a proporção real de formados que se tornam profissionais registrados.
